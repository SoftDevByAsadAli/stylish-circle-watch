# 🕒 Stylish Circle Watch

A modern, stylish, and responsive circle watch application built with **HTML, CSS, and JavaScript**. This project features a real-time clock, current date display, and dynamic greeting messages based on the time of day.

## ✨ Features
- **Real-time Clock:** Updates every second.
- **Dynamic Date:** Displays the current day and date.
- **Greeting System:** Automatically switches between "Good Morning", "Good Afternoon", and "Good Evening".
- **Responsive Design:** Looks great on all screen sizes.
- **Neon Glow Effect:** Stylish UI with hover animations.

## 🛠 Tech Stack
- HTML5
- CSS3 (Flexbox, Radial Gradients)
- JavaScript (Date Object, Intervals)

## 🚀 How to Run
1. Clone this repository.
2. Open `index.html` in your favorite web browser.
